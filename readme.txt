Create chat box on your website for customers to reach you.

== Description ==

With the AotolChat free plugin, creating a floating chat box on your website is easy. 
The plugin relies on the AotolChat mobile app so you can set up and customise the chat box and chat on the go. 
Receive message notifications via the app and get email reminders so you never miss an opportunity. 
AotolChat mobile app can be downloaded for free from [GooglePlay](https://play.google.com/store/apps/details?id=com.aotol.aotolchat "GooglePlay") or [AppStore](https://apps.apple.com/au/app/aotolchat/id1536058601 "AppStore").

== Frequently Asked Questions ==
= How to acquire AotolChat code =
You can find your AotolChat code inside AotolChat mobile app, available at https://company.aotol.com/aotolchat/
At the following are the steps
1. Download our AotolChat mobile app from GooglePlay(https://play.google.com/store/apps/details?id=com.aotol.aotolchat) or AppStore(https://apps.apple.com/au/app/aotolchat/id1536058601)
2. Launch the mobile app on your phone.
3. After you signed in the app, scroll to "Advanced function" and expend it.
4. Tap on the script code displayed under "Advanced function" to send the code to your email.
5. Copy and paste the code in the email to the input box below then save it.
